## 🚀 How to Open

### Option 1 — Direct Browser (Simple)
1. Extract the ZIP file
2. Double-click `index.html`
3. It opens in your default browser ✅

### Option 2 — Chrome File (Recommended)
1. Extract the ZIP file
2. Open **Google Chrome**
3. Drag and drop `index.html` into Chrome
   — OR —
   Press `Ctrl+O` → browse to `index.html` → Open
